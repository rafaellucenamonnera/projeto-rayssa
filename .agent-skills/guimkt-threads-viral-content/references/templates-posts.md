# Templates de Post — Referência Completa

Posts são conteúdo curto (single post, até 500 caracteres).
Cada template combina Hook + Insight em um único bloco.

---

## TEMPLATE P1 — One-Liner: "STOP/START"

### Formato
```
SÉRIO.

PARE de [erro comum que pessoas do nicho cometem]
e COMECE a [solução simples pro erro].

É isso.
```

### Regras
- Tom: Assertivo, sem enrolação, empoderador
- 2-3 linhas máximo
- Sem emojis (a estrutura e linguagem carregam o impacto)
- O contraste STOP vs START cria ritmo emocional

### Variações
- Versão com profanidade: incluir "porra" ou equivalente como intensificador
- Versão sem profanidade: estrutura idêntica, sem o intensificador
- Versão suave: trocar "PARE" por "Considere parar" (para nichos conservadores)

### Exemplo
```
SÉRIO.

PARE de otimizar campanha pelo CTR
e COMECE a otimizar pelo LTV do lead.

É isso.
```

## TEMPLATE P2 — One-Liner: Reversão

### Formato
```
Você não [comportamento/resultado indesejado].

Você [comportamento/resultado desejado].
```

### Regras
- Tom: Direto, inteligente, reflexivo
- 2 linhas apenas, cada uma com menos de 12 palavras
- Sem emojis, sem pontuação extra
- Os dois verbos/ações devem ser intercambiáveis

### Exemplo
```
Você não gera leads.

Você constrói relacionamentos que convertem.
```

## TEMPLATE P3 — One-Liner: "É tóxico"

### Formato
```
[Mito/crença/prática comum] é tóxico.

Normalize [método alternativo não-convencional].
```

### Regras
- Tom: Bold, corretivo, confrontacional
- 2-4 linhas máximo, cada uma com menos de 15 palavras
- Nomear algo aceito como problemático provoca engajamento imediato

### Variações de intensidade
- Máxima: "[X] é tóxico pra caralho."
- Média: "[X] é tóxico."
- Suave: "[X] está fazendo mais mal do que bem."

### Exemplo
```
"Postar todo dia" é tóxico.

Normalize postar com estratégia 3x por semana e ter resultado real.
```

## TEMPLATE P4 — One-Liner: Autoridade + Verdade

### Formato
```
Eu estudei [tema] / Sou [título] e preciso te contar:

[Afirmação bold e provocadora sobre o tema].

[Exemplos de comportamentos que comprovam o claim].

[Frase final curta que fecha o ponto].
```

### Regras
- Tom: Honesto, provocativo, fundamentado em cuidado
- 4-6 linhas máximo
- A abertura "Eu estudei / Sou" estabelece autoridade
- "Preciso te contar" cria tensão e curiosidade

### ⚠️ Regra ética
A autoridade declarada ("Eu estudei X" / "Sou X") DEVE ser real.
Se o usuário não tem a credencial, use a versão sem autoridade:
"Depois de [X] anos trabalhando com [Y], percebi que..."

### Exemplo
```
Trabalho com CRO há 10 anos e preciso te contar:

A maioria das landing pages não falha por design ruim.

Falha porque ninguém testou a headline.
Ninguém rodou um heatmap.
Ninguém perguntou pro lead o que ele queria.

Conversão é ciência, não achismo.
```

## TEMPLATE P5 — One-Liner: "REAL TALK"

### Formato
```
REAL TALK:

[Verdade impactante ou conselho acionável em 1 frase].
```

### Regras
- Tom: Real, não-filtrado, honesto, levemente vulnerável
- "REAL TALK:" sempre em caixa alta, linha própria
- 1 frase bold após (máx 15 palavras)
- Sem emojis. As palavras carregam o peso
- Funciona melhor com estrutura "X não é Y. É Z."

### Exemplo
```
REAL TALK:

Cobrar barato não é humildade. É insegurança disfarçada de estratégia.
```

## TEMPLATE P6 — One-Liner: Rejeição + Redefinição

### Formato
```
Não quero [meta superficial/status symbol].

Quero:
→ [Ação generosa específica para alguém]
→ [Ação que remove uma ansiedade real]
→ [Ação que cria uma memória ou experiência]

Isso é [conceito real que importa].
```

### Regras
- Tom: Sincero, quietamente poderoso, fundamentado
- Rejeitar meta superficial comum para criar pattern disruption
- 3 bullets com ações específicas e emocionais
- Fechar com redefinição em 1-3 palavras

### Exemplo
```
Não quero ser famoso no LinkedIn.

Quero:
→ Que meus clientes indiquem meu trabalho sem eu pedir
→ Dormir sabendo que o resultado do mês tá garantido
→ Ter tempo de jogar com meu filho sem olhar o celular

Isso é sucesso de verdade.
```

## TEMPLATE P7 — Listicle Rápido

### Formato
```
[Número] coisas que eu queria ter sabido sobre [tema] [tempo] atrás:

1. [Insight acionável]
2. [Insight acionável]
3. [Insight acionável]
...
7. [Insight acionável]

Faz isso e [resultado] mais rápido que quem não faz.
```

### Regras
- 5-10 itens (ímpares performam melhor)
- Cada item: 1 frase, máx 10 palavras
- Ordenar do mais curto pro mais longo (ritmo visual)
- Fechar com frase de impacto que cria urgência real

### Exemplo
```
7 coisas que eu queria ter sabido sobre CRO 5 anos atrás:

1. Teste o headline primeiro
2. Formulário curto sempre ganha
3. Prova social com números específicos
4. Mobile primeiro, desktop depois
5. Velocidade de carregamento mata conversão silenciosamente
6. O melhor CTA é aquele que diz exatamente o que acontece ao clicar
7. Dados de heatmap valem mais que opinião do designer e do CEO juntos

Aplica essas 7 e seus números mudam em 30 dias.
```

## TEMPLATE P8 — Listicle + CTA

Idêntico ao P7, mas adiciona bloco de CTA após a frase final:

```
[... listicle completo ...]

Aplica essas [N] e [resultado].

➡ Me siga pra mais [tema] direto e sem enrolação.
[CTA da oferta, se houver]
```

## TEMPLATE P9 — Provocação: Diagnóstico

### Formato
```
[Coisa 1] = [Realidade negativa] vendida como [Promessa positiva]
[Coisa 2] = [Realidade negativa] vendida como [Promessa positiva]
[Coisa 3] = [Realidade negativa] vendida como [Promessa positiva]
...

[Conceito central] é [metáfora master] de [problema do nicho].
```

### Regras
- 5-10 equações no formato X = Y vendido como Z
- Tom: Diagnóstico, filosófico, bold
- Cada linha é autocontida e punchline por si só
- Fechar com metáfora que redefine todo o problema
- Sem emojis no corpo. Máximo 1 na metáfora final

### Exemplo
```
"Postar todo dia" = burnout vendido como consistência
"Compra meu curso" = PDF vendido como transformação
"Escala com tráfego" = gastar mais vendido como crescer
"Funil automático" = spam vendido como automação
"6 em 7" = sorte vendida como método

Marketing de mentira é o cavalo de Tróia do mercado digital brasileiro.
```

## TEMPLATE P10 — Transformação Antes/Depois

### Formato
```
Eu com [idade/momento anterior]:
→ [Ponto negativo 1]
→ [Ponto negativo 2]
→ [Ponto negativo 3]

Eu com [idade/momento atual]:
→ [Resultado positivo 1]
→ [Resultado positivo 2]
→ [Resultado positivo 3]

Muita coisa muda em [período]. Foco.
```

### ⚠️ Regra ética
TODOS os dados (idades, momentos, pontos negativos e positivos)
DEVEM ser reais e fornecidos pelo usuário. Se não forneceu, PERGUNTE.

### Regras
- Tom: Honesto, bold, motivacional sem ser clichê
- 3 pontos em cada seção (simetria visual)
- Pontos negativos devem ser específicos, não genéricos
- Pontos positivos devem ser tangíveis, não abstratos
- Fechar com frase curta (< 10 palavras)
