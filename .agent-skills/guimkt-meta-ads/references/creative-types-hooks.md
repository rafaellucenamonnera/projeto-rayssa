## Hook Categories (resumo)

| Hook Type | Exemplos |
|-----------|----------|
| **Before/After** | Transformação após usar o produto |
| **Us/Them** | Nosso jeito vs. jeito antigo |
| **Problem/Solution** | Identificar problema, mostrar solução |
| **Offer** | Desconto, frete grátis, garantia, bundle |
| **Fact/Stat** | Número que causa impacto |
| **Credibility** | Reviews, prêmios, certificações, PR |

## Emotion Triggers

| Emotion | Quando usar |
|---------|-------------|
| Fear | Público precisa agir para evitar algo |
| Trust | Mercado novo, preço alto, decisão de risco |
| Joy | Produto de prazer, lifestyle |
| Anger | Público frustrado com status quo |
| Surprise | Resultado inesperado, diferencial forte |
| Hope | Transformação possível |
| Curiosity | Público frio, precisa parar o scroll |
| Belonging | Comunidade, "junte-se a nós" |
| Relation | "Isso sou eu" — identificação |
| Awe | Resultado impressionante, superioridade |
| Guilt | "Por que eu ainda não comprei?" |
| Entertained | Conteúdo nativo, UGC-style |
