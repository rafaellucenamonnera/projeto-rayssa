✓ Copy dentro dos limites de caracteres
✓ CTAs claros e orientados à conversão
✓ Descrições visuais detalham cena, paleta e estilo
✓ Adaptações para 3+ posicionamentos por conceito

Compliance Meta Ads

✓ Regra 20% texto respeitada em todos os conceitos
✓ Sem promessas exageradas ou garantias absolutas
✓ Sem conteúdo restrito (saúde, finanças sem disclaimer)
✓ Linguagem respeitosa e inclusiva

Diversidade de Conceitos

✓ Mínimo 3 ângulos diferentes

✓ Mix de estáticos + dinâmicos
✓ Cobertura de topo, meio e fundo de funil
