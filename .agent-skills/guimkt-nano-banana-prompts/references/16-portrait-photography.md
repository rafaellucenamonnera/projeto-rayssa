## Professional Headshots

### Corporate Headshot

**Descrição:** Retrato profissional para uso corporativo.

```json
{
  "subject": {
    "description": "Professional headshot of [PERSON DESCRIPTION]",
    "expression": "confident, approachable, slight smile",
    "face": {
      "preserve_original": true,
      "makeup": "natural, professional, well-groomed"
    }
  },
  "photography": {
    "camera_style": "professional studio headshot",
    "angle": "slight 3/4 turn, eyes to camera",
    "shot_type": "head and shoulders",
    "aspect_ratio": "3:4 or 1:1"
  },
  "lighting": {
    "type": "professional studio lighting",
    "setup": "butterfly/paramount lighting with fill",
    "quality": "soft, even, flattering"
  },
  "background": {
    "type": "solid neutral",
    "color": "light grey or subtle gradient",
    "depth": "shallow DOF separating subject"
  }
}
```

```json
{
  "subject_identity": {
    "face_match": "100% match to reference image",
    "body_proportions": "maintain exact reference proportions"
  },
  "pose_and_styling": {
    "posture": "High-fashion editorial slouch, confident",
    "gaze": "Directed towards camera"
  },
  "wardrobe_composition": {
    "ensemble_type": "[Describe full outfit]",
    "styling": "[How worn - open, draped, tight, etc.]"
  },
  "environmental_context": {
    "setting": "Minimalist architectural space",
    "background_surface": "Solid [color] wall",
    "lighting_conditions": {
      "source": "[Natural/Studio] light",
      "characteristics": ["High contrast", "warm highlights"]

  "photographic_execution": {
    "genre": "Stylish Editorial Fashion Portrait",

```

## Creative Portraits

### Dramatic Lighting Portrait

**Descrição:** Retrato com iluminação dramática e mood intenso.

```json
{
  "Objective": "Generate ultra-realistic cinematic portrait with dramatic lighting",
  "Subject": {
    "Expression": "Intense, emotionally charged gaze"
  },
  "Lighting": {
    "Style": "Dramatic Rembrandt or split lighting",
    "Key_Lights": ["Hard key light from side", "Minimal fill"],
    "Contrast": "High contrast with deep shadows"
  },
  "Camera": {
    "Shot_Type": "Close-up or medium close-up",
    "Lens": "85mm portrait lens",
    "Depth_of_Field": "Shallow DOF"
  },
  "Environment": {
    "Background": "Dark, minimal, non-distracting",
    "Mood": "Intense, mysterious, powerful"
  }
}
```

## Template Base para Retratos

```json
{
  "subject": {
    "description": "[One sentence describing person and action]",
    "age": "[approximate age range]",
    "expression": "[emotional state, gaze direction]",
    "hair": {"color": "[color]", "style": "[style with realistic details]"},
    "clothing": {
      "top": {"type": "[item]", "color": "[color]", "details": "[fit, fabric]"},
      "bottom": {"type": "[item]", "color": "[color]", "details": "[style]"}
    },
    "face": {
      "preserve_original": true,
      "makeup": "[level and style appropriate to context]"
    }
  },
  "accessories": {
    "jewelry": {"earrings": "[style]", "necklace": "[style]"},
    "eyewear": {"type": "[if applicable]"}
  },
  "photography": {
    "camera_style": "[style reference]",
    "angle": "[camera angle]",
    "shot_type": "[framing]",
    "aspect_ratio": "[ratio]"
  },
  "background": {
    "setting": "[environment]",
    "elements": ["[visible elements list]"],
    "lighting": "[lighting quality and source]"
  }
}
```
