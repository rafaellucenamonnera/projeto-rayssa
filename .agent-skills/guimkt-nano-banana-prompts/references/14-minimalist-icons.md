# ⚡ Minimalist Icons

> 7+ prompts curados para ícones minimalistas e design de interface.

---

## Prompts Principais

### App Icon Design

**Descrição:** Ícone de aplicativo em estilo moderno minimalista.

```
Minimalist app icon for [APP CONCEPT] on solid [COLOR] background.

Design elements:
- Single simple geometric shape or symbol
- Flat design with subtle gradient
- Clean rounded corners (iOS style)
- 3D subtle depth with soft shadows

Technical: 1024x1024 export-ready, no text, centered symbol.
```

---

### Icon Set Design

**Descrição:** Conjunto de ícones com estilo consistente.

```
Cohesive icon set of [NUMBER] icons for [CATEGORY/THEME].

Style parameters:
- Line weight: consistent [thin/medium/bold]
- Corner radius: uniform throughout
- Color: [monochrome/duotone/colorful]
- Grid: 24x24 or 48x48 base grid

Icons to include: [list of icon concepts]

Aesthetic: Modern SaaS / Apple-like / Material Design
```

---

### Symbol/Glyph Design

**Descrição:** Símbolo ou glifo único e memorável.

```
Minimal symbolic glyph representing [CONCEPT].

Design requirements:
- Works at small sizes (16px+)
- Single continuous form when possible
- Negative space utilization
- Scalable vector-ready appearance

Applications: App icon, favicon, watermark, brand mark.
```
