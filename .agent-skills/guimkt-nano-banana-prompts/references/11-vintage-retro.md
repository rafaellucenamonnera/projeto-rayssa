## Prompts Principais

### Apollo 11 Technical Blueprint

**Descrição:** Poster vintage com blueprints técnicos e render 3D.

```
An expert [DISCIPLINE] designer's presentation board for [SUBJECT] — 
[ICONIC FEATURES / ERA], featuring:

- Black-and-white 2D technical drawings with annotations and dimensions (LEFT)
- Exploded axonometric diagram revealing [KEY INTERNAL COMPONENTS] (CENTER)  
- Photorealistic 3D render of [SUBJECT] in [ICONIC ENVIRONMENT] (RIGHT)

Visual style transitions from technical/archival tones to 
emotional/atmospheric color palette. Clean grid layout, 
museum-grade industrial design presentation.

Title block: "[TITLE] — [YEAR / VARIANT / TAGLINE]"

Examples:
- Apollo 11 Lunar Module LM-5 "Eagle"
- Concorde SST
- Titanic RMS
- Model T Ford
```

**Fonte:** Repositório Nano Banana

### Historical Date Recreation

**Descrição:** Recriação fotorrealista de evento histórico.

```json
{
  "prompt": {
    "positive": "Location: {{LOCATION}} Date: {{YYYY-MM-DD}}. First search reliable sources and identify the most important historical event on this date at this location. Then generate a single photo-realistic, ultra-detailed, cinematic frame representing this event. Period accuracy mandatory: architecture, clothing, weapons/vehicles and city texture must be historically consistent. No modern objects, buildings, vehicles or signs should be visible. Single scene, single moment, real camera physics, natural human proportions, high micro detail.",
    "negative": "modern buildings, cars, asphalt, neon, smartphones, wrong era clothing/armor, fantasy, anime, cartoon, text overlay, blurry, low-res, extra limbs"
  },
  "render": {
    "quality": "ultra",
    "resolution": "4k"
  }
}
```

```

Style: 90s software packaging aesthetic, bold typography, 
feature bullets, system requirements box, shrink-wrapped look.
```
