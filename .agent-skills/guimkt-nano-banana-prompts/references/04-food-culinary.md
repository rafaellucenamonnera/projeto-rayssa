## Exploded View & Infographics

### Taco Ingredients Breakdown

**Descrição:** Ingredientes de taco separados e alinhados verticalmente com anotações.

```
Exploded view of taco, presented as clean, commercial recipe-style breakdown.
Exactly five ingredients, separated and arranged vertically from top to bottom, 
evenly spaced and perfectly aligned.

Ingredient order (top → bottom):
- Fresh tomato salsa — 40 g
- Shredded cheddar cheese — 30 g
- Grilled chicken pieces — 80 g
- Crisp lettuce — 25 g
- Soft wheat taco tortilla — 60 g (bottom base)

Add clear infographic-style annotations for each ingredient.
Each annotation includes ingredient name and exact weight in grams.

Annotation design guidelines:
– Clean sans-serif font, medium weight
– Text placed inside minimal frames or boxes
– Thin, precise connector lines pointing directly to each ingredient
– High readability, no overlap, no decorative excess
– Structured vertical layout, like a modern recipe card

Background is light, neutral, optimized for text clarity.
Style: minimal, instructional, commercial.
```

**Fonte:** [@Strength04_X](https://x.com/Strength04_X/status/2006031630268522943)

### Culinary Atlas (4 Países)

**Descrição:** Grid 2x2 mostrando variações culturais do mesmo ingrediente em formato de livro aberto.

```
Inputs: Ingredients = [User Choice, e.g., Chicken, Bacon]

Goal: A "Culinary Atlas" Grid showing 4 different cultural variations.

Layout Rule:
- Output Format: A 2x2 Grid.
- Content per Panel: Each of 4 panels shows a Single Open Book Spread.

Inside Each Panel (The Book Spread Rule):
- Left Page (The History): 2D Vintage Sepia Sketches. Visual timeline 
  or illustrated recipe of the dish. Flat, ink-drawn style.
- Right Page (The Reality): 3D Pop-up Diorama. The finished, steaming 
  meal emerges physically from the page. A tiny miniature chef from 
  that country is standing on the page next to the massive food.

Consistency: All 4 books should look like part of the same series, 
but with different cultural contents.

Lighting: Cinematic top-down lighting, highlighting depth difference 
between flat left page and 3D right page.

Output: 2x2 Grid, High Definition, Macro Detail.
```

**Fonte:** [@Gdgtify](https://x.com/Gdgtify/status/2004986136335110526)

### Mona Lisa Charcuterie Board

**Descrição:** Obra de arte recriada como tábua de charcuterie com ingredientes.

```
Act as a world-class food expert and artist.

Input A: Masterpiece painting (e.g., Mona Lisa, Starry Night).

Analyze composition, colors, mood, elements. Pick the right food 
ingredients relevant to the region that can be used in the painting.

Goal: Large charcuterie platter replicating the painting using 
cured meats, cheeses, fruits, nuts, and ingredients relevant to region.

Rules: Decide which ingredients should be used for each part, 
so the painting looks as realistic as possible but is mainly made of food.

2x2 grid: 
- Panel 1: Original painting reference
- Panel 2: Ingredient map
- Panel 3: Overhead build
- Panel 4: Final board with wine pairing

Output: 1 x 1:1 image, rustic gourmet photography.
```
