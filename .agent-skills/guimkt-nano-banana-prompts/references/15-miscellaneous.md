# ✨ Miscellaneous

> 4+ prompts curados para estilos diversos e experimentais.

---

## Prompts Principais

### Infographic Design

**Descrição:** Design de infográfico visual educacional.

```
Educational infographic about [TOPIC].

Layout: Vertical flow with clear sections
Content: [3-5 key points with visual representations]

Design elements:
- Clean iconography for each point
- Data visualization where applicable
- Clear hierarchy with headers and subheaders
- Consistent color scheme throughout

Style: Modern corporate / Editorial / Playful
Background: Light neutral or brand color
```

---

### Comparison Grid

**Descrição:** Grid comparativo de produtos ou conceitos.

```
Side-by-side comparison grid of [ITEMS TO COMPARE].

Layout: [2x2 / 3x3 / horizontal strip] grid
Each cell contains: [item image + key details]

Consistent across all cells:
- Same lighting
- Same angle
- Same background
- Same info panel layout

Style: Commercial product comparison, catalog-ready.
```

---

### Abstract Pattern

**Descrição:** Padrão abstrato para backgrounds ou texturas.

```
Seamless abstract pattern featuring [ELEMENTS].

Style: [Geometric / Organic / Digital / Hand-drawn]
Color palette: [specific colors or mood]
Scale: [Large bold / Medium detail / Fine texture]

Application: Background tile, fabric print, packaging.
Seamless tiling required for production use.
```
