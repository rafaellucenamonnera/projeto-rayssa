## Fase 1: ICP — Exemplo

**Cliente:** SaaS de gestão para indústrias

| Dimensão | Descrição |
|----------|-----------|
| **Faixa Etária** | 35 a 55 anos |
| **Profissão** | Gestão industrial, engenharia de produção |
| **Cargo** | Diretor de operações, gerente de produção, CEO de PME industrial |
| **Setor** | Indústria de transformação, metalurgia, plásticos, alimentos |
| **Formação** | Engenharia, administração (pós-graduação em gestão) |
| **Objetivos** | Reduzir custos operacionais, aumentar eficiência produtiva, digitalizar processos |
| **Dores** | Controle manual de produção, perdas por retrabalho, falta de visibilidade em tempo real |
| **Necessidades** | Sistema integrado de gestão, dashboards de produção, rastreabilidade de lotes |
| **Tópicos de Interesse** | Indústria 4.0, lean manufacturing, automação industrial, ERP industrial |

**Perfil Psicográfico:**
- Critérios de decisão: ROI comprovado, cases no mesmo setor, suporte técnico local
- Nível de consciência: Consciente do problema, buscando soluções ativamente
- Objeções comuns: Custo de implantação, resistência da equipe, integração com sistemas legados
- Canais preferidos: Google Search, LinkedIn, feiras industriais, indicações
